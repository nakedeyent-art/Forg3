## Immediate Brand-Use Rule

Use `FORG3(TM)` or `FORG3(SM)` now. Do not use `FORG3(R)` until the USPTO registration actually issues. USPTO guidance allows TM/SM before registration and reserves the registration symbol for federally registered marks.

## Recommended Trademark Filings

### Filing 1: Standard Character Word Mark

- Mark: `FORG3`
- Format: Standard character mark
- Why first: This protects the word/number brand regardless of font, color, logo layout, or future redesign.
- Filing basis:
  - Use in commerce, Section 1(a), if the app is already publicly offered in U.S. commerce with a real specimen.
  - Intent to use, Section 1(b), if the public commercial launch is not yet live.
- Owner: `NAK3D EYE ENTERPRISES` or the exact legal entity/person that owns Forg3. Confirm exact legal name before filing.

Recommended initial classes:

- International Class 42: Software as a service featuring software for electronic signatures, secure document execution, document routing, audit logs, and sealed document downloads.
- International Class 9: Downloadable mobile applications for electronic signatures, secure document execution, document routing, audit logs, and sealed document downloads.

Optional later class after attorney review:

- International Class 45 may be relevant only if Forg3 provides legal/document-authentication services beyond software tooling. Do not add this unless the service description is accurate.

## Copyright Registration

The logo artwork can be registered with the U.S. Copyright Office as visual art if there is enough human-authored original expression. Copyright exists when original visual art is fixed, but registration creates a public record and gives additional enforcement benefits.

Recommended registration path:

- Office: U.S. Copyright Office electronic registration portal.
- Work type: Visual Arts.
- Title: `Forg3 Secure E-Signatures Logo`
- Deposit: final flat logo artwork, not only the photographed mockup.
- Claimant: same owner as the logo copyright holder.
- Author: the actual human designer if human-designed, or the entity if it was a work made for hire and the paperwork supports that.
- Limitation note if AI was used: disclose and exclude any non-human-generated material; claim only human-authored selection, arrangement, edits, or final design contributions. Do not represent fully AI-generated material as fully human-authored.

Key ownership questions to answer before filing:

1. Who created the logo?
2. Was it created by an employee, contractor, AI tool, or design service?
3. If a contractor/designer made it, is there a signed copyright assignment or work-made-for-hire agreement?
4. Was any stock art, template, font, icon, or AI-generated element used?
5. Has this logo already been published publicly? If yes, first publication date and country.
